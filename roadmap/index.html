Roadmap

Where we're headed

Up next In progress

Features actively being built for the next release.

  • Onboarding A guided first-run experience to get new users set up quickly.
  • Location types Categorise spots as skatepark, street spot, bowl, DIY, and more.
  • Location attributes Tag spots with surface material, coverage, lighting, and other details that affect skateability.
  • Location photos Add photos to locations so others know what to expect before they visit.
  • Advanced filtering Filter saved locations by type, attributes, and current score.
  • Search improvements Faster, more relevant results when finding existing spots to save.
  • Map view improvements Pan and zoom the map to explore locations near any point, not just your current position.
Requested Community

Features requested by the community. No timeline yet, but all are on our radar.

  • Notifications Get alerted when a saved spot dries out or when conditions are forecast to be good for a session.
  • Auto session detection Use location detection to prompt a session log when you arrive at a saved spot, so you never forget to report.
  • Eyes on the prize Log a weather observation when you pass a spot but aren't skating it. Helps the community know conditions without a full session report.
  • More photos Support for multiple photos per location, including community-contributed shots.
  • Parking data Show nearby parking options and notes on each location page.
  • Save scheduled sessions Plan sessions in advance by scheduling a time and spot, so you can coordinate with others and keep track of upcoming skates.
  • Get notified about changes Receive alerts when a saved location's details are updated — such as changes to surface, obstacles, or access — so your information stays current.
